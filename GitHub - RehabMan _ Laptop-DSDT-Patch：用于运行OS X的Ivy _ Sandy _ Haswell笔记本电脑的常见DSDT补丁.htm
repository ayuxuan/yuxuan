README.md

RehabMan的笔记本补丁库

这套补丁可以在DSDT编辑器中使用，或者（最好是）MaciASL在笔记本电脑上运行一些常用的补丁来运行OS X。这些补丁对于Sandy Bridge或Ivy Bridge笔记本电脑来说是很常见的。

其中许多补丁程序来自HP ProBook补丁程序，或者在我自己的英特尔DH67GD桌面上运行。

许多电池补丁是随着时间的推移而创建的，在tonymacx86.com上以我个人的职位发布，并在此处收集。

将这些补丁添加到MaciASL作为存储库：

  • 运行MaciASL
  • 从MaciASL菜单栏中选择首选项
  • 选择来源
  • 点击[+]按钮
  • 给它一个名字（例如“笔记本电脑补丁”）
  • 键入以下URL：http：//raw.github.com/RehabMan/Laptop-DSDT-Patch/master
